# Industrial Coating Price List

This table lists the cost-per-unit-area for each coating type used in the AquaShield protection system.
Use the Coating Code found in the scan file attributes to lookup the corresponding price.

| Coating Code | Coating Name | Cost ($/cm²) | Description |
| :--- | :--- | :--- | :--- |
| **5** | Noise Artifact | 0.50 | Low-confidence scan debris / ghost echoes. |
| **12** | Basic Epoxy | 3.25 | Standard anti-corrosion epoxy layer. |
| **37** | Marine Grade Polyurethane | 8.75 | Heavy-duty marine polyurethane coating. |
| **60** | Ceramic Thermal Barrier | 12.00 | High-temperature ceramic spray coating. |
| **88** | Titanium Nitride | 15.50 | Ultra-hard wear-resistant coating. |

## Formula

$$ \text{Total Cost} = \text{Surface Area} \times \text{Cost Per Unit Area} $$

Note: Ensure surface area is in $cm^2$ and cost is in $/cm^2$.
